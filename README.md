## Proyecto prueba SensoWave

A partir de un fichero json dar una propuesta de código donde se visualicen los datos de animales y si es posible agrupados por categoría de sexo.

### Resultado:

El proyecto se ha realizado con React, podéis ver el resultado en el siguiente enlace: https://claudiapalazon.github.io/sensowave-animals/

- Una web donde se visualizan los datos en forma de tabla. Hay un filtro para poder agrupar los resultados por la categoría de sexo (todos, hembras o machos).
- Dentro de la tabla se puede seleccionar cada uno de los resultados por si se quisiera ver un detalle más específico de cada uno de ellos.
